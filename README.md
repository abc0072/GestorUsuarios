## Introducción 

La interfaz muestra una pantalla llamada Gestor de Usuarios con un encabezado superior que incluye el título y un icono. En la parte izquierda aparece un menú lateral con botones para navegar a Dashboard, Usuarios, Informes, Ajustes y Ayuda. En el centro se encuentra un formulario donde se pueden introducir o editar los datos de un usuario: nombre, email, rol (Admin, Editor o Invitado), un checkbox para indicar si está activo y un área de texto para añadir notas. A la derecha hay un panel con dos pestañas: una de Resumen, que muestra un texto descriptivo no editable, y otra de Logs, que contiene una lista de registros. En la parte inferior de la pantalla hay tres botones de acción: Cancelar, Limpiar y Guardar. Todo está presentado con fondo azul oscuro, texto blanco y botones en azul medio. <br>
Al pulsar el botón de guardar, aparecera un cuadro que mostrará el mensaje "¿Guardar cambios?", y dos botones, uno de aceptar y tro de cancelar.
